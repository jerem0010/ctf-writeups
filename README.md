# CTF Writeups

A collection of my Capture The Flag writeups, mostly focused on cryptography, smart contract security, reverse engineering, and low-level exploitation.

The goal of this repository is not only to store solutions, but to explain the reasoning process behind each challenge: how I analyzed the problem, what vulnerability I found, how I exploited it, and what I learned from it.

## Focus Areas

- Cryptography
- RSA
- Elliptic curves
- ECDSA
- Commitments
- Hashing
- Smart contract security
- Solidity / EVM
- Reverse engineering
- Binary exploitation
- Algorithms

## Writeup Format

Most writeups follow this structure:

```txt
# Challenge Name

- Category
- CTF
- Difficulty / points
- Status

## Challenge

What we are given.

## Analysis

How I approached the problem.

## Vulnerability

The core weakness.

## Exploit

How the solve script works.

## Flag

Final result.

## Lessons Learned

What this challenge taught me.
```

## Why I Write These

Writing CTF solutions forces me to:

- understand the vulnerability deeply
- explain the attack clearly
- build reproducible exploits
- improve my cryptography and security reasoning
- keep track of techniques for future audits and research
